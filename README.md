PyNotes is an advanced text-editor made in python. Run the installpynotes.sh script to install the latest version automatically. Otherwise to install a specific version, use:
./installpynotes.sh {version}. Eg:
./installpynotes.sh 1.3.9
This needs super-user privileges and an internet connection, as it downloads and installs the deb package from github automatically.
IMPORTANT: You can only install PyNotes version 1.2.7 and above with this script, as the older versions had a completely different file structure.
